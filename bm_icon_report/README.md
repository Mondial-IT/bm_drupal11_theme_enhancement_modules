## BM Icon Report

Lists discovered icon CSS classes for developers.

- Scans custom themes (and bm_main css) for CSS files with icon selectors (`icon`, `gv-icon`, `flaticon`, `bm-` with `:before/:after content`).
- Displays sets with icon class, glyph, and code.
- Refresh link adds a status message and reloads the catalogue.
- Available under Reports â†’ Icon report (`/admin/reports/icon-report`).

Notes:
- Attaches the CSS sources it finds so glyphs render on the report page.
- Uses bm_main/icon styling library for layout.
